# c-grundkurs

Uppgift 15 Logical error 1: Man måste byta på if checken mindre än märket till större än så att man får största talet och inte minsta talet i arrayn.

Uppgift 21 Logical error 2: Man måste skriva koden som frågar åldern, sedan måste man lägga if statementen så att de är från största numret till det minsta för att om din ålder är 50 så kommer den säga att du är över 18 som stämmer men det kommer att stanna där och inte forsätta till den rätta.

Uppgift 22 Logical error 3: Man måste ta och byta i for loopen <= till bara < för att annors går det -1 som är inte i arrayn som gör att programmet inte fungerar.
